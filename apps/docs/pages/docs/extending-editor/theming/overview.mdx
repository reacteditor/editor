# Theming

<Callout type="info">
  Theming in React Editor is currently limited in functionality.
</Callout>

The [Theming API](/docs/api-reference/theming) supports:

- [Fonts](fonts): Change the font family, or font loading mechanism.

For further changes to the React Editor interface, use the [composition](../composition) and [UI overrides](../ui-overrides) APIs.
