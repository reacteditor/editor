# Introduction

Welcome to the React Editor documentation!

## What is React Editor?

React Editor is a modular, open-source visual editor for React.js. You can use React Editor to build custom drag-and-drop experiences with your own application and React components.

Because React Editor is just a React component, it plays well with all React.js environments, including Next.js. You own your data and there's no vendor lock-in.

React Editor is also licensed under MIT, making it suitable for both internal systems and commercial applications.

## Main Features

| Feature                                                                   | Description                                                                                                          |
| ------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| [Component Configuration](/docs/integrating-editor/component-configuration) | Integrate your own components with React Editor by providing render functions and configuring fields that map to your props. |
| [Root Configuration](/docs/integrating-editor/root-configuration)           | Customize the root component that wraps all other React Editor components.                                                   |
| [Multi-column Layouts](/docs/integrating-editor/multi-column-layouts)       | Create multi-column layouts using nested components. Now supports advanced CSS layouts.                              |
| [Categories](/docs/integrating-editor/categories)                           | Group your components in the side bar.                                                                               |
| [Dynamic Props](/docs/integrating-editor/dynamic-props)                     | Dynamically set props after user input and mark fields as read-only                                                  |
| [Dynamic Fields](/docs/integrating-editor/dynamic-fields)                   | Dynamically set fields based on user input                                                                           |
| [External Data Sources](/docs/integrating-editor/external-data-sources)     | Load content from a third-party CMS or other data source                                                             |
| [Server Components](/docs/integrating-editor/server-components)             | Opt-in support for React Server Components                                                                           |
| [Data Migration](/docs/integrating-editor/data-migration)                   | Migrate between breaking React Editor releases and your own breaking prop changes                                            |
| [Viewports](/docs/integrating-editor/viewports)                             | Preview and edit your content in a same-origin iframe to simulate different viewports                                |
| [Feature Toggling](/docs/integrating-editor/feature-toggling)               | Toggle React Editor features, like duplication or deletion, via the permissions API.                                         |
