# useGetEditor

A hook that returns a function that retrieves the latest [`EditorApi`](/docs/api-reference/editor-api) at call time. This is useful in callbacks, effects, or any logic that runs outside the component render lifecycle.

```tsx copy
import { useGetEditor } from "@frontendai/react-editor";

const Example = () => {
  const getFrontend = useGetEditor();

  const handleClick = useCallback(() => {
    // Current EditorApi is always provided
    const { appState } = getFrontend();
  }, [getFrontend]);

  return <button onClick={handleClick}>Click me</button>;
};
```

If you need to react to changes to `EditorApi`, try the [`useEditor` hook](use-editor).

## Returns

A function to retrieve the latest [`EditorApi`](/docs/api-reference/editor-api) data at call time.
