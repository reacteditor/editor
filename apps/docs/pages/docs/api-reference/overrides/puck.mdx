# editor

Override the React Editor children. This is the equivalent of passing in [`children`](/docs/api-reference/components/editor#children) to the [`<Editor>`](/docs/api-reference/components/editor) component.

```tsx copy
const overrides = {
  editor: ({ children }) => <div>{children}</div>,
};
```

## Props

| Prop                    | Example   | Type      |
| ----------------------- | --------- | --------- |
| [`children`](#children) | `<div />` | ReactNode |

### `children`

The default node for the [`<Editor>`](/docs/api-reference/components/editor) children.
