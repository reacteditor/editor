# Data Model

The React Editor data model

- [AppState](data-model/app-state) - React Editor's internal state.
- [ComponentData](data-model/component-data) - The data model for each component instance.
- [Data](data-model/data) - The data model produced by React Editor for a page.
- [RootData](data-model/root-data) - The data model for the root data.
