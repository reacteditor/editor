# \<Editor.Preview\>

Render a drag-and-drop preview for the current data when composing a custom React Editor UI.

<EditorPreview
  config={{
    components: {
      HeadingBlock: {
        fields: {title: {type: 'text'}},
        render: ({title}) => <div>{title}</div>
      }
    }
  }}
  data={{
    root: { props: {} },
    content: [{ type: "HeadingBlock", props: { id: "HeadingBlock-123", title: 'Hello, world' } }],
  }}
>

  <Editor.Preview />
</EditorPreview>

```tsx {17} showLineNumbers copy
import { Editor } from "@reacteditor/core";

export function Editor() {
  return (
    <Editor>
      <Editor.Preview />
    </Editor>
  );
}
```

## Props

| Param       | Example                    | Type   | Status |
| ----------- | -------------------------- | ------ | ------ |
| [`id`](#id) | `id: "my-preview-content"` | String | -      |

## Optional props

### `id`

A unique identifier for the preview frame. Default: `editor-preview`.

```tsx {6} copy
import { Editor } from "@reacteditor/core";

export function Editor() {
  return (
    <Editor>
      <Editor.Preview id="my-frame" />
    </Editor>
  );
}
```
