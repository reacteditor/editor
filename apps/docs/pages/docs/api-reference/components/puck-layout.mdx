# \<Editor.Layout\>

Render the standard Frontend layout. This is useful for implementing behavior using the [internal Frontend API](/docs/extending-editor/internal-editor-api) without changing the default UI.

```tsx {13} showLineNumbers copy
import { Editor } from "@frontendai/react-editor";

const Example = ({ children }) => {
  useEffect(() => console.log("Hello, world"), []);

  return <>{children}</>;
};

export function Editor() {
  return (
    <Editor>
      <Example>
        <Editor.Layout />
      </Example>
    </Frontend>
  );
}
```

## Props

This component doesn't accept any props.
