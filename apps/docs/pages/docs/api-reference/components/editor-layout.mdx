# \<Editor.Layout\>

Render the standard React Editor layout. This is useful for implementing behavior using the [internal React Editor API](/docs/extending-editor/internal-editor-api) without changing the default UI.

```tsx {13} showLineNumbers copy
import { Editor } from "@reacteditor/core";

const Example = ({ children }) => {
  useEffect(() => console.log("Hello, world"), []);

  return <>{children}</>;
};

export function Editor() {
  return (
    <Editor>
      <Example>
        <Editor.Layout />
      </Example>
    </Editor>
  );
}
```

## Props

This component doesn't accept any props.
