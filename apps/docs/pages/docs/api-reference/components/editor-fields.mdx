# \<Editor.Fields\>

Render the fields for the currently selected item in [`<Editor.Preview>`](/docs/api-reference/components/editor-preview) when composing a custom React Editor UI.

<EditorPreview
  config={{
    components: {
      HeadingBlock: {},
    },
  }}
  data={{ root: { props: {} }, content: [] }}
  style={{ padding: 4 }}
>
  <Editor.Fields />
</EditorPreview>

```tsx {14} showLineNumbers copy
import { Editor } from "@reacteditor/core";

export function Editor() {
  return (
    <Editor>
      <Editor.Fields />
    </Editor>
  );
}
```

## Props

| Param                       | Example             | Type    | Status |
| --------------------------- | ------------------- | ------- | ------ |
| [`wrapFields`](#wrapfields) | `wrapFields: false` | boolean | -      |

## Optional props

### `wrapFields`

Whether or not the top-level fields should be padded and separated by a a border. Defaults to `true`.

```tsx {6} copy
import { Editor } from "@reacteditor/core";

export function Editor() {
  return (
    <Editor>
      <Editor.Fields wrapFields={false} />
    </Editor>
  );
}
```
