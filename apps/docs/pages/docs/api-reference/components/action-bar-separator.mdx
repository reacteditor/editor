# \<ActionBar.Separator\>

Render a separator in the [`<ActionBar>`](action-bar). An alternative to [`<ActionBar.Group>`](action-bar-group) that avoids wrapping elements.

```tsx showLineNumbers {3} copy
<ActionBar>
  <ActionBar.Label label="Label 1" />
  <ActionBar.Separator />
  <ActionBar.Label label="Label 2" />
  <ActionBar.Action>★</ActionBar.Action>
</ActionBar>
```
