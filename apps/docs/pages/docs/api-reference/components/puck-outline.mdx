# \<Editor.Outline\>

Render an interactive outline of the current data payload when composing a custom Frontend UI.

<EditorPreview
  config={{
    components: {
      HeadingBlock: { render: () => <div>Hello world</div> },
      ParagraphBlock: {},
    },
  }}
  data={{
    root: { props: {} },
    content: [{ type: "HeadingBlock", props: { id: "HeadingBlock-123" } }],
  }}
>
  <Editor.Outline />
  {/* Outline doesn't render unless Preview is rendered due to zone flushing */}
  <div style={{ display: "none" }}>
    <Editor.Preview />
  </div>
</EditorPreview>

```tsx {12} showLineNumbers copy
import { Editor } from "@frontendai/react-editor";

export function Editor() {
  return (
    <Editor>
      <Editor.Outline />
    </Frontend>
  );
}
```

## Props

This component doesn't accept any props.
