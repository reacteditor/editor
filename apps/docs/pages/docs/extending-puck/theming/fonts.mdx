# Fonts

Frontend uses the [Inter typeface family](https://rsms.me/inter/) by default, loaded via a CDN.

It's possible to change the font, or provide your own version of Inter if you need to host it locally.

## Load your own font file

To load your own font file, use the `no-external.css` bundle instead of the primary one.

```css
/* @import "@frontendai/react-editor/frontend.css"; */
@import "@frontendai/react-editor/no-external.css";
```

## Changing the font family

To change the font family Frontend uses, change the [`--editor-font-family`](/docs/api-reference/theming#--editor-font-family) CSS property:

```css
/* @import "@frontendai/react-editor/frontend.css"; */
@import "@frontendai/react-editor/no-external.css";

:root {
  --editor-font-family: Arial;
}
```
